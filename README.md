# Guia de Informações — COA Relatórios

Base de conhecimento interativa e documentação completa de todos os visuais dos relatórios Power BI do Centro de Operações Agroindustriais (COA) da Tereos.

## 📍 Acesso

🔗 **Link da plataforma:** https://OtavioFrasato.github.io/coa-guia-relatorios

## 🎯 Objetivo

Esta plataforma foi desenvolvida para servir como **base de conhecimento acessível** para todos os usuários dos Power BIs do setor COA Relatórios, independente do nível técnico.

Cada visual documentado inclui:
- ✅ O que o visual mostra (explicação simples)
- ✅ Origem dos dados (tabelas, schemas, fontes)
- ✅ Cálculos DAX explicados
- ✅ Como interpretar os dados
- ✅ Pontos de atenção e avisos
- ✅ Print do visual real
- ✅ Tags para busca rápida

## 📊 Relatórios documentados

1. **Relatório de Manutenção** — Ordens de serviço, backlog e desempenho
2. **CTT** — Centro de Trabalho e Transferência
3. **Indústria** — Produção de açúcar e etanol
4. **Perda de Ritmo** — Análise de perdas e paradas
5. **Treminhado** — Logística de transporte de cana
6. **Desempenho** — KPIs consolidados
7. **Agroindustrial** — Qualidade de cana e ATR
8. **Preparo** — Operações de preparo de solo
9. **Plantio** — Monitoramento de plantio
10. **Tratos Culturais** — Aplicações agrícolas

## 🔍 Como usar

- **Busca rápida:** Pressione `Ctrl+K` para ativar a busca
- **Navegação:** Use a sidebar para selecionar relatórios
- **Expandir visual:** Clique no card do visual para ver todas as informações

## 🛠️ Desenvolvimento

- **Frontend:** HTML5, CSS3, JavaScript puro
- **Design:** Dark theme com identidade visual Tereos/COA
- **Data:** JSON estruturado (sem banco de dados)
- **Hospedagem:** GitHub Pages
